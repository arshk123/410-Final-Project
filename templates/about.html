    About Us

    This website was created for our CS410 final project at the University of Illinois. It uses Pitchfork ratings and performs sentiment analysis on the review text in order to assign ratings to each artist in our database. It also allows you to select your own ratings for artists, even ones not rated by pitchfork, and uses those ratings in order to recommend you new artists through a process called Collaborative Filtering.

    Technologies/Libraries Used

    Surprise VADER Sentiment Analysis Library Spotify API Unofficial Pitchfork API Unofficial Billboard API Flask PostgreSQL
